void GAMMA_Counts ( void )

Function: void GAMMA_Counts(void)

Summary: Function to track the counts from the gamma detector.

Description: Function to track the counts from the gamma detector. When a transition from the gamma detector is detected, a flag is set in system interrupt. One gamma count causes two interrupts. This is accounted for. The count is then transferred to the REPORT array for later transmission to the host.
