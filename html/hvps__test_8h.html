File Name: hvps_test.h

Summary: This header file provides prototypes and definitions for hvps_test.c.

Description: This header file provides function prototypes and data type definitions for hvps_test.c. Some of these are required by the system (such as the "HVPS_TEST_Tasks" prototypes) and some of them are only used internally by HVPS_TEST_Tasks (such as the "Set_HVPS_Ramp_Rate" definition). Both are defined here for convenience.
