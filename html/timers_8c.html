Detailed Description

Company: Hunting Titan

File Name: timers.c

Summary: To provide timer functions to various parts of the program.

Description: This file contains the state machine code for the TIMERS operation. It implements the logic of the TIMERS state machine. It provides timer functions to various parts of the program. This machine uses TIMER instance 1. It has been configured to provide a 100uS interrupt. This is then used to generate clocks at 100uS, 1mS, 10mS, 100mS and 1000mS. Various task ticks are generated as needed.
