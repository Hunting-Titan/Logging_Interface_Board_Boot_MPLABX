void Set_HVPS_Ramp_Rate ( uint16_t value)

Function: void Set_HVPS_Ramp_Rate( uint16_t value )

Summary: This function sets update rate for the HVPS control voltage.

Description: This function sets update rate for the HVPS control voltage. Value is a rate in volts/second. This value is then converted to an update rate for the control DAC. For example, an update value of 100V/second gives an DAC update rate of approx. 20mS. This update rate is used with Timer 1 to control when the control DAC is updated.

Precondition: None.

Parameters: uint16_t value: Value representing volts/second.

Returns None.

Remarks: None.

Example: Set_HVPS_Ramp_Rate( uint16_t value )
