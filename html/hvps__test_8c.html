Detailed Description

Company: Hunting Titan

File Name: hvps_test.c

Summary: This file contains the source code for testing the (HVPS) wire-line shooting power supply.

Description: This file contains the source code for testing the (HVPS) wire-line shooting power supply. It implements the logic of the HVPS_Test_Tasks state machine. This machine is called from a command. It performs a test of the HVPS to ensure proper operation. It drives the HVPS control voltage and ensures the output of the HVPS is within tolerance. The DMPLED is turned on if the test fails. The states are configured so that other system tasks can be maintained while these tasks are waiting for an event to occur.
