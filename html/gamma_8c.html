Company: Hunting Titan

File Name: gamma.c

Summary: This file contains the state machine code for the GAMMA operation.

Description: This file contains the state machine code for the GAMMA operation. Once started, the GAMMA detects a change in the gamma detector line. The count is incremented and the amount updated once per second. Each second starts a new count.
