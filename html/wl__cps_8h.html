Detailed Description

Company: Hunting Titan

File Name: wl_cps.h

Summary: This header file provides prototypes and definitions for wl_cps.c.

Description: This header file provides function prototypes and data type definitions for wl_cps.c. These functions are used to monitor the wire-line communication power supply voltage and current, to set the voltage and to turn the supply on and off. Both the current and voltage are checked for over limit conditions and will set flags to turn the communication power supply (WL_CPS) off.

void Adjust_WL_CPS_Voltage ( uint8_t target)

Function: void Adjust_WL_CPS_Voltage(uint8_t target)

Summary: This function adjusts the WL_CPS to the desired voltage.

Description: This function adjusts the WL_CPS to the desired voltage. The value is checked to ensure it is within the adjustable limits of 15 to 64 VDC. That voltage is converted to a table value that contains the appropriate pot settings for a given voltage. The pot settings is then transmitted via SPI port to the WL_CPS control pot.

Precondition: None.

Parameters: uint8_t target - desired voltage between 15 and 64V.

Returns None.

Remarks None.

Example: Adjust_WL_CPS_Voltage(30)
