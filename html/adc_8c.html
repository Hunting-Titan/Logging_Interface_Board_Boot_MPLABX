Detailed Description

Company: Hunting Titan

File Name: adc.c

Summary: This file contains the state machine code for the ADC operation.

Description: This file contains the state machine code for the ADC operation. Once started, the ADC operation is governed by the a 1mS timer using Timer Instance 4. The ADC results are captured in the ADC interrupt routine and flags are set so the values can be transferred to the appropriate functions for later use. After the timer and ADCs are started, this machine is set to idle.

void ADC_Tasks ( void )

Function: void ADC_Tasks(void)

Summary: ADC tasks function

Description: This routine is the ADC tasks function. It defines the ADC's state machine and core logic. This machine initializes the Timer and ADCs used to digitize signal inputs. Once the timer and ADCs are started, this machine is set to idle and the the updated ADC values are retrieved in the system interrupt routine.

Precondition: The system ("SYS_Initialize") and application ("APP_Initialize")initialization should be called before calling this.

Parameters: None.

Returns: None.

Example: ADC_Tasks()

Remarks: This routine must be called from the APP_Tasks() routine.
