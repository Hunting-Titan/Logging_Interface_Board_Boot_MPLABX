Function:

void CF_Tasks(void)

Summary: CF_Tasks function

Description: This routine is the CF_Tasks function. It defines the CF_Tasks state machine and core logic. This machine is used to monitor the wire-line shooting power supply (CF) voltage and current and will set flags when those limits are exceeded. Those flags are monitored by the HV_PS machine and will turn the supply off if either of those flags are set.

Precondition: The system ("SYS_Initialize") and application ("APP_Initialize")initialization should be called before calling this.

Parameters: None.

Returns: None.

Remarks: This routine must be called from the APP_Tasks() routine.

Example: CF_Tasks()
