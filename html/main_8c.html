File Name: main.c

Summary: This file contains the "main" function for the shooting power supply panel.

Description: This file contains the "main" function for the shooting power supply panel. The "main" function calls the "SYS_Initialize" function to initialize the state machines of all MPLAB Harmony modules in the system and it calls the "SYS_Tasks" function from within a system-wide "super" loop to maintain their correct operation. These two functions are implemented in configuration-specific files (usually "system_init.c" and "system_tasks.c") in a configuration-specific folder under the "src/system_config" folder within this project's top-level folder. The APP_Initialize and APP_Tasks functions are implemented in app.c and contain the application specific files.
