Detailed Description

Company: Hunting Titan

File Name: switches.h

Summary: This header file provides prototypes and definitions for switches.c.

Description: This header file provides function prototypes and data type definitions for switches.c. These functions are used to monitor the front panel switches and actuate relays based on the switch settings. Some of these are required by the system (such as the "APP_Initialize" and "APP_Tasks" prototypes) and some of them are only used internally by the application (such as the "SWITCH_STATES" definition). Both are defined here.

◆ read_switches()

static void read_switches ( void )
static

Function: static void read_switches(void)

Summary: This function determines all of the switch positions.

Description: This function determines all of the switch positions. It does this by monitoring all of a given switches lines and ensuring that only one line per switch is active. For switches with multiple positions, each position is put into an array. Only one location in a given array can be active or true. The switch locations are then passed on to be processed and converted into commands to actuate relays or other activity.

Precondition: None.

Parameters: None.

Returns None.

Remarks: None.

Example: read_switches()
