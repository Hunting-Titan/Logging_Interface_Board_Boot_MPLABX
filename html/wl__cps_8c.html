Detailed Description

Company: Hunting Titan

File Name: wl_cps.c

Summary: This file contains the source code for controlling and monitoring the wire-line communications power supply.

Description: This file contains the source code for controlling and monitoring the wire-line communications power supply. It implements the logic of the WL_CPS's state machine. The states are configured so that other system tasks can be maintained while these tasks are waiting for an event to occur. The machine monitors the communication wire line current and voltage and will turn the supply off if an overcurrent or overvoltage condition occurs. Additional functions support commands and operations as needed.
