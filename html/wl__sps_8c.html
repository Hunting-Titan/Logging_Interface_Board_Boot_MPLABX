Detailed Description

Company: Hunting Titan

File Name: wl_sps.c

Summary: This file contains the state machine code for monitoring the wire-line shooting power supply (WL_SPS) operation.

Description: This file contains the state machine code for monitoring the wire-line shooting power supply (WL_SPS) operation. It implements the logic of the WL_SPS state machine The states are configured so that other system tasks can be maintained while these tasks are waiting for an event to occur. The machine monitors the wire line current and voltage and will set flags used be the HV_PS machine to turn the supply off.

void WL_SPS_Tasks(void)

Summary: WL_SPS_Tasks function

Description: This routine is the WL_SPS_Tasks function. It defines the WL_SPS_Tasks state machine and core logic. This machine is used to monitor the wire-line shooting power supply (WL_SPS) voltage and current and will set flags when those limits are exceeded. Those flags are monitored by the HV_PS machine and will turn the supply off if either of those flags are set.

Precondition: The system ("SYS_Initialize") and application ("APP_Initialize")initialization should be called before calling this.

Parameters: None.

Remarks: This routine must be called from the APP_Tasks() routine.
