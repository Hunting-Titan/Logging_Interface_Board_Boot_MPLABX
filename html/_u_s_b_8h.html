Summary: This header file provides prototypes and definitions for USB.c.

Description: This header file provides function prototypes and data type definitions for USB.c. Some of these are required by the system (such as the "APP_Initialize" and "APP_Tasks" prototypes) and some of them are only used internally by the application (such as the "USB_TASKS" definition).
