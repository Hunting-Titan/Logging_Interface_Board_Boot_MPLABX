void SYSTEM_OKAY_Tasks ( void )

Function: void SYSTEM_OKAY_Tasks(void)

Summary: SYSTEM_OKAY tasks function

Description: This routine is the SYSTEM_OKAY tasks function. It defines the SYSTEM_OKAY's state machine and core logic. This code should execute only if the fuse does not blow when the SYSTEM_OKAY timer expires. This code will turn off all power
supplies so the tool can be safely retrieved.

Precondition: The system ("SYS_Initialize") and application ("APP_Initialize")initialization should be called before calling this.

Parameters: None.

Returns: None.

Remarks: This routine must be called from APP_Tasks() routine.

Example: SYSTEM_OKAY_Tasks()
