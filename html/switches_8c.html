Detailed Description

Company: Hunting Titan

File Name: switches.c

Summary: This file contains the state machine code for the SWITCH task operation.

Description: This file contains the state machine code for the SWITCH task operation and its associated functions. It implements the logic of the SWITCH tasks state machine.
This function monitors when the STATUS button is pressed. It does this by monitoring the switch line. After the switch is pushed, a count is incremented to track the number of times the switch is pushed. After ten pushes, the count is reset. The switch count is sent to the GAMMA Gain function to update the gain setting for the first stage gamma count circuit. The states are configured so that other system tasks can be maintained while these tasks are waiting for an event to occur.
