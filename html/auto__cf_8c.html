Detailed Description

Company: Hunting Titan

File Name: auto_cf.c

Summary: This file contains the state machine code for the AUTO_CF operation.

Description: This file contains the state machine code for the AUTO_CF operation. Once started, the AUTO_CF operation uses a 50 sample current array (approximately 50mS time window)to look for a negative going current spike that is greater than the spike threshold. This spike is an indication when a detonator has ignited. A flag is set to indicate an ignition and the machine is set to idle to wait until the auto ramp command completes. Afterward, the machine is set to respond to another auto ramp command. The states are configured so that other system tasks can be maintained while these tasks are waiting for an event to occur.

Summary: AUTO_CF tasks function

Description: This routine is the AUTO_CF tasks function. It defines the AUTO_CF's state machine and core logic. This machine tracks the wire line current when an HVPS auto ramp command is given and indicates when a current spike is detected.

Precondition: The system ("SYS_Initialize") and application ("APP_Initialize")initialization should be called before calling this.

Parameters: None.

Returns: None.

Example: AUTO_CF_Tasks()

Remarks: This routine must be called from the APP_Tasks() routine.
