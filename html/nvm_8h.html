Detailed Description

Company: Hunting Titan

File Name: nvm.h

Summary: This header file provides prototypes and definitions for nvm.c.

Description: This header file provides function prototypes and data type definitions for nvm.c. Some of these are required by the system (such as the "APP_Initialize" and "APP_Tasks" prototypes) and some of them are only used internally by the application (such as the "TIMER_Tasks" definition). Both are defined here for convenience.
